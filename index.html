What is Wine?

Wine (originally an acronym for “Wine Is Not an Emulator”) is a compatibility layer capable of running Windows applications on several POSIX-compliant operating systems, such as Linux, Mac OS X, & BSD. Instead of simulating internal Windows logic like a virtual machine or emulator, Wine translates Windows API calls into POSIX calls on-the-fly, eliminating the performance and memory penalties of other methods and allowing you to cleanly integrate Windows applications into your desktop.

Why CrossOver?

CrossOver, while composed primarily of WINE open-source core, also contains proprietary “value-add” components. CrossOver gives you the ability to run multiple INDEPENDENT copies of the Notes Client, Domino Administrator, and Domino Designer simultaneously — all while using separate Notes IDs — with much lower overhead and licensing costs than any available virtualization technology. CrossOver does not require an underlying Windows license and provides less CPU overhead, and therefore a much longer battery life.

How to Install?

Register to join the mailing list and see Designer and Administrator run natively on Mac and Linux!

Once you received the Email with the download links, proceed with the following steps:

  1. Unzip, Launch and move CrossOver to Application folder if prompted
  2. Create a new Windows XP Bottle by clicking on the bottles icon at the top left of CrossOver, then click the + icon at the bottom left and select New Bottle.
  3. Configure the newly created Bottle:
    1. Right-click the created Bottle and enable Legacy X Windows from theSettings submenu
    2. Reboot the Bottle when prompted
    3. Run Command “regedit”
    4. Navigate to HKEY_CURRENT_USER/Software/Wine/X11 Driver/
    5. In the right panel, right-click and create a new String Value and name it“ClientSideGraphics”
    6. Right-click on the newly created key and modify the value to “N”
  4. Install the prerequisite applications:
    1. Install Windows Application, scroll to “Runtime Support Components”dropdown
    2. Select “Microsoft Visual C++ 6.0” and Proceed
    3. Select your newly created Bottle and finish the installation process
  5. Repeat the above steps for the following:
    1. Platform SDK Redistributable: GDI+
    2. Internet Explorer 7. Install the XML Parser when prompted
    3. Flash Player 11
  6. Install IBM Notes Software:
    1. Click Install Windows Application
    2. Select Other Application from the Unsupported Applications dropdown list
    3. Select “Choose installer File” browse to your Notes Client (Designer and Administrator) Installer
  7. (If you are experiencing graphics artifacts Launch Wine Configuration and checkEmulate Virtual Desktop in the Graphics tab).
